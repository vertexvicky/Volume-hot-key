# VolumeHotKey-Windows

## 📌 Description
VolumeHotKey-Windows is a simple tool that allows you to control your system volume using hotkeys. It is useful for PCs that do not have dedicated volume control keys.

### 🔥 Features:
- **Increase Volume**: `Win + Page Up`
- **Decrease Volume**: `Win + Page Down`
- **Mute/Unmute Volume**: `Win + End`

This script runs in the background and listens for these key combinations to adjust your system's volume.

## 🛠️ How It Works
This project consists of a **Python script** that listens for global hotkeys using the `keyboard` module and sends system volume commands via the Windows API.

### 🔑 **Hotkey Functions**
| Hotkey | Action |
|--------|--------|
| `Win + Page Up` | Increase volume |
| `Win + Page Down` | Decrease volume |
| `Win + End` | Mute/Unmute |

Once installed, the script will run silently in the background and respond to these hotkeys.
